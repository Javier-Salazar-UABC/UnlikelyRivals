\doxysubsection{Descripción detallada}
Inspector de propiedades para editar objetos y tilemaps en tiempo real. 

Panel Im\+Gui que permite editar propiedades de objetos seleccionados, incluyendo transformación, sprites, shaders, controles y trayectorias. También permite edición de propiedades de tilemaps.

Utiliza múltiples sub-\/paneles para diferentes tipos de componentes:\+
\begin{DoxyItemize}
\item Transform:\+ posición, velocidad, rotación
\item Sprite:\+ textura, dimensiones, escala
\item Shader:\+ uniforms del shader
\item Control:\+ estado de botones de entrada
\item Paths:\+ editor de trayectorias (delegado a \doxylink{classCE_1_1GPaths}{GPaths})
\end{DoxyItemize}

\begin{DoxyNote}{Nota}
Derivada de \doxylink{classCE_1_1GLayer}{GLayer} para integración en sistema GUI del motor 

Contiene instancia de \doxylink{classCE_1_1GPaths}{GPaths} para edición de trayectorias 
\end{DoxyNote}
