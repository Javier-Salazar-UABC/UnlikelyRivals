\doxysubsection{Descripción detallada}
Panel de Im\+Gui para listar y seleccionar objetos de la escena. 

Renderiza una lista de todos los objetos en el pool actual permitiendo seleccionar uno para editar sus propiedades. Integra el inspector de propiedades (\doxylink{classCE_1_1GPropiedades}{GPropiedades}) para edición en tiempo real.

\begin{DoxyNote}{Nota}
Derivada de \doxylink{classCE_1_1GLayer}{GLayer} para integración en sistema GUI del motor 

Contiene instancia embedded de \doxylink{classCE_1_1GPropiedades}{GPropiedades} para edición 
\end{DoxyNote}
