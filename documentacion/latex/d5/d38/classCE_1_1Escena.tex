\doxysubsection{Descripción detallada}
Clase base abstracta para las escenas del juego. 

Define la interfaz que deben implementar todas las escenas del juego. Una escena contiene lógica específica, objetos, tilemaps y mapeos de entrada.

Características:\+
\begin{DoxyItemize}
\item \doxylink{classCE_1_1Pool}{Pool} de objetos para gestión de entidades
\item Sistema de mapeo de botones a acciones
\item Soporte para tilemaps con múltiples capas
\item Hooks del ciclo de vida (init, update, render, cleanup)
\end{DoxyItemize}

\begin{DoxyNote}{Nota}
Las subclases deben implementar todos los métodos virtuales puros 
\end{DoxyNote}
