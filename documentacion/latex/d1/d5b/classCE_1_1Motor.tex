\doxysubsection{Descripción detallada}
Controlador principal del motor de juego que orquesta el ciclo de juego. 

Esta clase es responsable de:\+
\begin{DoxyItemize}
\item Inicializar el motor con la configuración proporcionada
\item Ejecutar el bucle principal de juego (game loop)
\item Procesar eventos de entrada del usuario
\item Actualizar la lógica del juego
\item Renderizar la escena actual
\item Gestionar las capas de interfaz gráfica (GUI)
\end{DoxyItemize}

El motor utiliza un patrón singleton implícito a través de la inyección de dependencias, permitiendo múltiples instancias pero típicamente una sola activa.
