\doxysubsection{Descripción detallada}
Panel de Im\+Gui para editar trayectorias/\+paths de entidades. 

Proporciona interfaz de edición visual para componentes \doxylink{classCE_1_1IPaths}{IPaths}, permitiendo crear y modificar curvas Bézier para que los objetos sigan trayectorias predefinidas.

\begin{DoxyNote}{Nota}
Derivada de \doxylink{classCE_1_1GLayer}{GLayer} para integración en sistema GUI del motor 

Trabajo en tándem con \doxylink{classCE_1_1GPropiedades}{GPropiedades} para selección de objetos 
\end{DoxyNote}
