\doxysubsection{Descripción detallada}
Clase base abstracta para capas de interfaz gráfica Im\+Gui. 

Define la interfaz mínima que deben implementar todas las capas GUI del motor. Las capas se apilan en un vector y se actualizan/\+renderizan secuencialmente cada frame.

Las capas típicas incluyen:\+ dock (información), logger (logs), viewport, propiedades de objetos, lista de objetos, etc.
