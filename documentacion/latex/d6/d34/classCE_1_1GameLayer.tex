\doxysubsection{Descripción detallada}
Clase base abstracta para la capa de lógica principal de juego. 

Define la interfaz que debe implementar la capa de juego, permitiendo que el motor gestione el ciclo de vida de la aplicación. Proporciona puntos de enganche (hooks) para inicialización, procesamiento de entrada, actualización y renderizado.

Las clases derivadas deben implementar:\+
\begin{DoxyItemize}
\item \doxylink{classCE_1_1GameLayer_a04ef1c3b4c271ed27aa74bb65edb7dff}{On\+Inputs()} en sus tres variantes para procesamiento de entrada flexible
\item \doxylink{classCE_1_1GameLayer_a8454bfacf849027ceaa2e904a47e54a8}{On\+Update(float dt)} para lógica de actualización
\item \doxylink{classCE_1_1GameLayer_a36785531dba2e09839a0387b8fa94d09}{On\+Render()} para renderizado (opcional) 
\end{DoxyItemize}
