\doxysubsection{Descripción detallada}
Gestor singleton de recursos (fuentes, texturas, sonidos, música). 

Implementa un cargador y caché de recursos multimedia del juego. Utiliza punteros compartidos para gestión automática de memoria y almacena recursos por clave textual para acceso fácil.

Soporta:\+
\begin{DoxyItemize}
\item Fuentes de texto (True\+Type)
\item Texturas de imágenes (PNG, JPG, BMP, etc)
\item Sonidos cortos (WAV, OGG, FLAC)
\item Música con loop automático (WAV, OGG, FLAC)
\end{DoxyItemize}
