\doxysubsection{Descripción detallada}
Clase base para objetos del juego implementando patrón de arquitectura basada en componentes. 

Hereda de sf::\+\+Drawable y sf::\+\+Transformable de SFML para integración con el sistema de renderizado. Utiliza composición de componentes (\doxylink{classCE_1_1IComponentes}{IComponentes}) para definir comportamiento y propiedades.

Características principales:\+
\begin{DoxyItemize}
\item Sistema de componentes genéricos adjuntables en runtime
\item Detección dinámica de componentes por tipo (RTTI)
\item Contador estático de objetos activos
\item Componentes principales preestablecidos:\+ Nombre, Transformación, Estadísticas
\end{DoxyItemize}

\begin{DoxyNote}{Nota}
Las subclases deben implementar \doxylink{classCE_1_1Objeto_a7018b9059b915f5cbd8220e0a1ffe6c3}{on\+Update(float dt)} para lógica de actualización 
\end{DoxyNote}
