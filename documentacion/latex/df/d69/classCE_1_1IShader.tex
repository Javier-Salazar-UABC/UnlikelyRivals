\doxysubsection{Descripción detallada}
Componente para aplicar efectos de shader (GLSL) a una entidad. 

Gestiona un shader SFML con uniforms variables. Permite establecer parámetros del shader como colores, tiempos, vectores y matrices.

\begin{DoxyNote}{Nota}
Los uniforms se almacenan como void\texorpdfstring{$\ast$}{*} con tipo erasure para flexibilidad 
\end{DoxyNote}
