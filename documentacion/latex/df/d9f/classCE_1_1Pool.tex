\doxysubsection{Descripción detallada}
Gestiona un pool de objetos del juego con tamaño dinámico. 

Implementa un contenedor para almacenar y acceder a objetos compartidos. Actúa como un administrador de recursos para una colección de entidades.

\begin{DoxyNote}{Nota}
El tamaño puede exceder max\+\_\+size dinámicamente (no es una restricción dura) 
\end{DoxyNote}
