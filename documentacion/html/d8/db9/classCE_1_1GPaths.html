Descripción detallada

Panel de ImGui para editar trayectorias/paths de entidades.

Proporciona interfaz de edición visual para componentes IPaths, permitiendo crear y modificar curvas Bézier para que los objetos sigan trayectorias predefinidas.

Nota
Derivada de GLayer para integración en sistema GUI del motor
Trabajo en tándem con GPropiedades para selección de objetos
