Descripción detallada

Inspector de propiedades para editar objetos y tilemaps en tiempo real.

Panel ImGui que permite editar propiedades de objetos seleccionados, incluyendo transformación, sprites, shaders, controles y trayectorias. También permite edición de propiedades de tilemaps.

Utiliza múltiples sub-paneles para diferentes tipos de componentes:

  • Transform: posición, velocidad, rotación
  • Sprite: textura, dimensiones, escala
  • Shader: uniforms del shader
  • Control: estado de botones de entrada
  • Paths: editor de trayectorias (delegado a GPaths)
Nota
Derivada de GLayer para integración en sistema GUI del motor
Contiene instancia de GPaths para edición de trayectorias
