Descripción detallada

Clase base abstracta para cámaras de juego que gestiona la vista.

Implementa una cámara que puede seguir objetos y está vinculada a un view de SFML. Proporciona transformación (posición, rotación) y métodos de actualización virtual para permitir diferentes estrategias de movimiento (fija, seguimiento, LERP, etc).

Nota
Utiliza weak_ptr para evitar referencias circulares con el objeto seguido
