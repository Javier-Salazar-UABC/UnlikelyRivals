Descripción detallada

template<typename T>
class CE::IManipularDatos< T >

Interfaz template para manipulación de datos en contenedores.

Define operaciones comunes para agregar y eliminar elementos de una colección, permitiendo eliminación por índice, desde el final (pop) y desde el inicio (pop front).

TTipo de datos de los elementos contenidos
