Descripción detallada

Clase para aritmética y geometría 2D.

Encapsula un vector 2D con operaciones comunes como magnitud, normalización, distancia y operadores algebraicos. Se integra con SFML permitiendo conversión desde/hacia sf::Vector2f.

Nota
Coordenadas públicas (x, y) para acceso directo
